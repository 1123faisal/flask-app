# Coffee Machine OOP Demo

Small Python OOP practice project that models a coffee machine system.

## Components

- `menu.py` for drink menu and pricing
- `coffee_maker.py` for resources and brewing checks
- `money_machine.py` for payments and reporting
- `main.py` for orchestration demo

## Run

```bash
uv run python main.py
```

or

```bash
python main.py
```

## Purpose

- Practice class design
- Practice collaboration between objects
- Simulate simple resource and payment workflows
