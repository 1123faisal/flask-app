# Project Quote Generator

Node.js script that generates a formatted quotation document using the `docx` library.

## Features

- Programmatic DOCX generation
- Styled sections, tables, headers, and footers
- Suitable for business quotation templates

## Stack

- Node.js (CommonJS)
- docx

## Run Locally

```bash
npm install
node demo.js
```

## Notes

- Update branding values and text in `demo.js` to match your company.
- Output file name and location are controlled in script logic.
